# lookup v1.2.7

## 수정 사항

- HWP 변환에서 원치 않는 가로줄이 보이는 문제를 줄이기 위해 자동 품질 판별을 추가했습니다.
- HWP 인쇄형 변환 옵션 적용 순서를 고정하고, 옵션 적용 스냅샷 진단 로그를 추가했습니다.
- HWP 변환 결과를 인쇄형/저장형 두 후보로 비교해 더 안정적인 결과를 자동 선택하도록 바꿨습니다.
- HWP 캐시 키에 품질 선택 버전을 반영하고 캐시 스키마를 갱신해 이전 잘못된 변환 결과 재사용을 줄였습니다.
- `document:open`, `document:convert` 응답에 `convertDiagnostics`를 추가해 선택 모드와 후보 점수를 함께 전달하도록 개선했습니다.
