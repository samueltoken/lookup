# lookup v1.2.8

## 수정 사항
- HWP 선택 엔진을 v2(균형형)로 개선했습니다.
  - print 변환을 먼저 빠르게 점검합니다.
  - 이상 징후가 있으면 saveas/full 비교를 수행합니다.
  - 필요할 때만 hwpjs fallback까지 확장합니다.
- 특정 HWP에서 보이던 원치 않는 줄 문제를 줄이기 위해 품질 판별을 강화했습니다.
  - 기존 벡터 기반 점수에 rasterLineScore(가로줄 관통 패턴 추정)를 추가했습니다.
  - 최종 점수는 vector+raster 통합 점수로 선택합니다.
- HWP 열기 체감 속도를 개선했습니다.
  - 선택 메타 캐시(선택 모드/점수/파일 서명)로 재열기 시 재분석을 생략합니다.
  - COM 변환 실행을 비동기 경로로 바꿔 UI 멈춤을 줄였습니다.
- 진단 정보를 보강했습니다.
  - convertDiagnostics에 selectorVersion, analysisMode, 후보별 상세 점수를 포함했습니다.
