# lookup v1.2.5

## 수정 사항
- HWP 보안 안내 최소화
  - 한컴 COM 열기 전에 보안 모듈 등록 상태를 먼저 점검하고 자동 등록을 시도합니다.
  - 등록 불가 환경에서는 즉시 호환 경로로 넘어가 열기 흐름이 끊기지 않도록 보강했습니다.
- HWP 페이지 붙음(책자형) 완화
  - HWP 레이아웃 변환 결과가 비정상적으로 넓은 페이지(2-up)로 감지되면 자동 재변환을 1회 시도합니다.
- 보기 모드 확장
  - `연속 단면`, `연속 양면`, `현재 페이지만` 보기 모드를 추가했습니다.
  - 보기 모드는 저장되어 다음 실행에도 유지됩니다.
- XLSX 열람 강화
  - COM/LibreOffice 실패 시 텍스트 나열 대신 시트 표 HTML 변환 경로로 열람성을 높였습니다.
  - `EMPTY_DOCUMENT`, `ENGINE_MISSING`, `ENGINE_TIMEOUT`, `CONVERT_FAILED`를 분리 안내합니다.
- 전체화면/패널 토글 안정화
  - 공통 복구 루틴에서 현재 페이지 강제 렌더/중앙 정렬을 적용해 흰 화면 재발을 줄였습니다.
