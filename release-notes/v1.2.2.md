lookup v1.2.2

수정 사항
- 흰 화면 재발 차단
  - 패널 숨김을 `display:none` 대신 폭 0 처리로 바꿨습니다.
  - 문서 영역을 그리드 가운데 칸에 고정했습니다.
  - 전체화면/패널 토글/창 크기 변경에서 같은 복구 루틴을 사용합니다.
- 인쇄 기능 복구
  - 인쇄 버튼, Ctrl+P, 메뉴 인쇄를 실제 시스템 인쇄 대화상자로 연결했습니다.
  - 기존 미리보기 열기 경로는 보조 기능으로 유지했습니다.
- 설치 문구 안정화
  - 설치창 제목을 `lookup v1.2.2 업데이트` 맥락으로 유지합니다.
  - 한국어 문구를 `업데이트 중입니다. 잠시만 기다려주세요.`로 고정했습니다.
  - 빌드 전에 `messages.yml` UTF-8 검증을 추가했습니다.
- 리본 아이콘 개선
  - 이모지 아이콘을 단색 SVG 아이콘으로 교체했습니다.
  - 인쇄, 회전, 삭제, 화면 토글, 다크모드, 실행 취소/다시 실행 아이콘을 정리했습니다.
