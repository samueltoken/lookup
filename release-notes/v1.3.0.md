# lookup v1.3.0

## 주요 변경
- 한컴/오피스 엔진이 없는 환경에서도 문서를 읽을 수 있도록 변환 경로를 정리했습니다.
- 지원 확장자를 `ppt`, `pptx`까지 확장했습니다.
- HWP 변환 결과를 캐시로 재사용하기 전에 유효 PDF 검사(헤더/크기/페이지 수/파서 열기)를 반드시 통과하도록 강화했습니다.
- 깨진 변환 파일은 자동 폐기 후 다음 후보로 전환하도록 바꿔 `Invalid PDF structure` 흰 화면 재발을 줄였습니다.

## 변환/진단 개선
- 변환 상태 이벤트(`엔진 확인 → 변환 → 검사 → 표시`)를 추가해 진행 상황을 UI에서 확인할 수 있게 했습니다.
- `document:open`/`document:convert` 응답에 변환 진단 정보를 유지해 원인 분석이 쉬워졌습니다.
- HWP selector 캐시 재사용 조건을 강화하고, 검증 실패 캐시는 즉시 무효화합니다.
- Hancom COM 스크립트의 경로 배열 문법 오류를 정리해 native 후보 생성 안정성을 높였습니다.

## 파일 연결/설치
- 설치/연결 확장자에 `ppt`, `pptx`를 추가했습니다.
- 오리 아이콘 기반 연결 프로그램 등록을 유지하고, 설치 후 레지스트리 반영 범위를 확장했습니다.
