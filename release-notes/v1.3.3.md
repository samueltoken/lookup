# lookup v1.3.3

## HWP 열기 품질/안내 개선
- HWP 실패 화면에서 안내 흐름을 정리했습니다.
  - 다시 시도
  - 호환 보기로 열기
  - 진단 정보 복사
- 원본 변환 실패 시 내부 경로/스크립트 줄번호 같은 긴 기술 문자열이 화면에 그대로 나오지 않도록 정리했습니다.
- 호환 보기 재시도 동작을 별도 경로로 분리해, 같은 파일을 바로 다시 열 수 있게 개선했습니다.

## 변환 로딩 오버레이 개선
- 변환 오버레이 문구를 중복 없이 표시하도록 로직을 정리했습니다.
  - 제목: 한글 문서를 여는 중입니다.
  - 단계: 현재 변환 단계 문구
- 단계 진행에 맞춰 진행 막대(퍼센트)가 계속 갱신되도록 보강했습니다.
- 실패 시 빈 화면 대신 안내 오버레이가 유지되도록 보완했습니다.

## 업데이트 완료 팝업 가독성 개선
- 버전 표시는 목록과 분리된 배지로 유지합니다.
- 섹션 제목/상위 항목/하위 항목 들여쓰기 구조를 유지해 읽기 쉽게 정리했습니다.
- HTML/마크다운 잔여 문자가 노출되지 않도록 정제 규칙을 유지/보강했습니다.
