# lookup v1.2.4

## 수정 사항
- 줌 성능 개선
  - 보이는 페이지 중심 렌더로 바꿔 Ctrl+휠 중 깜빡임/스크롤 급변을 줄였습니다.
  - 입력 중 미리 확대, 입력 종료 후 고해상도 보정 렌더를 적용했습니다.
- 변환 캐시 개선
  - 캐시 키에 앱 버전/변환 모드/엔진 토큰을 포함해 오래된 실패 캐시 재사용을 줄였습니다.
  - fallback 캐시는 짧은 유효시간(TTL)으로 관리합니다.
- HWP/XLSX 분기 개선
  - HWP는 COM 결과를 우선 사용하고, 실패 시 fallback으로 전환합니다.
  - XLSX는 빈 문서 여부를 별도로 분리해 안내합니다.
- 연결 프로그램 아이콘 개선
  - 빌드 후 exe 아이콘 삽입 단계를 보강해 연결 프로그램 아이콘 일치성을 높였습니다.
