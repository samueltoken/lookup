# lookup v1.2.4

## 수정 사항
- 줌 성능 개선
  - Ctrl+휠 확대/축소 중 전체 페이지 재렌더를 제거했습니다.
  - 화면에 보이는 페이지와 주변 페이지만 우선 렌더하도록 변경했습니다.
  - 휠 입력 중에는 빠른 미리 확대를 적용하고, 입력이 멈춘 뒤 선명 렌더를 실행합니다.
- 문서 변환 파이프라인 보강
  - 변환 캐시 키에 앱 버전, 변환 모드, 엔진 토큰을 포함해 오래된 캐시 재사용을 줄였습니다.
  - HWP/HWPX는 한컴 COM 성공 시 fallback 캐시보다 우선해 새 PDF를 사용합니다.
  - XLS/XLSX는 빈 문서 여부를 먼저 점검해 실패와 빈 문서를 분리 안내합니다.
  - fallback 변환 결과는 짧은 유효시간 캐시로 관리합니다.
- 연결 프로그램 아이콘 개선
  - `lookup.exe` 아이콘 삽입을 빌드 후 단계에서 추가 적용하도록 보강했습니다.
  - 우클릭 연결 프로그램 목록에서 오리 아이콘이 유지되도록 설치 레지스트리 경로를 유지했습니다.
