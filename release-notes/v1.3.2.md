# lookup v1.3.2

## HWP 열기 품질 복구
- 원본 변환 우선 경로를 더 엄격하게 적용했습니다.
- 검증된 원본 변환 결과가 있으면 바로 열고, 실패하면 단계적으로 호환 보기로 전환합니다.
- 호환 보기일 때는 상단 안내를 고정 표시해 `원본과 배치가 다를 수 있음`을 명확히 보여줍니다.

### 세부 개선
- HWP 호환 텍스트 정제를 강화해 `~~`, 불필요 기호, 과도한 취소선 표시를 줄였습니다.
- hwpjs HTML 생성이 실패해도, 마크다운 기반 호환 레이아웃 HTML을 만들어 표/목록/문단 가독성을 높였습니다.
- 내부 경로(ps1 파일 경로, 워커 줄 번호 등) 같은 진단 문자열은 사용자 화면에 노출하지 않도록 정리했습니다.

## HWP 로딩 UI 정리
- 변환 오버레이를 `제목 1줄 + 현재 단계 1줄 + 진행 막대 + 보조 문구` 형태로 고정했습니다.
- 같은 문장이 제목/단계에 중복 표시되던 문제를 제거했습니다.
- 단계형 진행률(`엔진 확인 -> 열기 -> 변환 -> 검사 -> 표시`)이 즉시 반영되도록 개선했습니다.

## 업데이트 완료 팝업 가독성 개선
- 버전 표시는 목록과 분리된 배지로 유지합니다.
- 섹션 제목/상위 항목/하위 항목 들여쓰기를 분리해 읽기 쉽게 표시합니다.
- HTML/마크다운 잔여 기호가 그대로 보이지 않도록 정제 규칙을 보강했습니다.
