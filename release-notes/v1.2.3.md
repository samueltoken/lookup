# lookup v1.2.3

## 수정 사항
- 문서 변환 엔진 안정화
  - HWP/HWPX는 한컴 COM 우선, 실패 시 호환 경로로 자동 전환합니다.
  - DOC/DOCX/XLS/XLSX는 Office COM 실패 시 LibreOffice/fallback 경로로 전환합니다.
- 패널/전체화면 전환 시 페이지 가시성 복구를 강화했습니다.
- 인쇄 경로를 하나로 통일해 버튼/단축키/메뉴 동작 차이를 제거했습니다.
- 리본 하단 보조 문구를 줄이고 아이콘 가독성을 개선했습니다.
