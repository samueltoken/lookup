lookup v1.2.3

수정 사항
- 문서 변환 안정화
  - HWP/HWPX 변환에서 한컴 COM 열기 방식을 보강했습니다.
  - DOC/DOCX/XLS/XLSX는 Office COM 실패 시 LibreOffice 변환을 자동 시도합니다.
  - 변환 실패 시 이유를 더 쉬운 문장으로 안내합니다.
- 화면 표시 안정화
  - 패널 토글/전체화면/리사이즈 복구 루틴에서 페이지 가시성 보장을 강화했습니다.
  - 전체화면의 문서 바깥 배경을 흰색 계열로 조정했습니다.
- 인쇄 복구
  - 인쇄 버튼, Ctrl+P, 메뉴 인쇄를 시스템 인쇄 대화상자 경로로 통일했습니다.
  - 기본 인쇄 경로 실패 시 대체 인쇄 경로로 자동 전환합니다.
- 아이콘 및 리본 정리
  - 미리보기, 되돌리기, 다시하기 아이콘을 더 얇고 단순한 SVG로 교체했습니다.
  - 리본 하단 그룹 설명 문구를 기본 숨김으로 바꿨습니다.
- 연결 프로그램 아이콘 개선
  - 실행 파일 아이콘 반영 설정과 레지스트리 보강으로 연결 프로그램 목록 아이콘 인식을 개선했습니다.
