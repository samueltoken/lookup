# lookup v1.2.6

## 수정 사항
- HWP 줄 문제 완화
  - HWP 기본 변환 경로를 `한컴 COM 인쇄형 PDF 출력`으로 고정했습니다.
  - 인쇄형 프로파일에서 검토/메모/표시선 계열 출력 옵션을 비활성화하도록 보강했습니다.
  - 인쇄형 출력 실패 시에만 `SaveAs` fallback으로 내려가도록 순서를 정리했습니다.
- 검색 하이라이트 정밀도 개선
  - 검색용 문자열을 원문 인덱스와 매핑해 단어 위치 좌표를 더 정확히 계산하도록 변경했습니다.
  - 하이라이트 박스를 단어 내부 밴드 형태로 보정해 단어 바깥 칠해짐을 줄였습니다.
  - 검색 결과 데이터에 `rects` 좌표를 포함해 클릭 이동/표시 좌표계를 통일했습니다.
- 도구/설정 동선 정리
  - 주석 기능(`형광펜/펜/텍스트 메모/되돌리기/다시하기`)을 `도구` 탭으로 통합했습니다.
  - `업데이트 확인/개발자 문의/언어 설정/버전 정보`는 상단 `설정` 메뉴에서 바로 실행되도록 정리했습니다.
- XLSX 시트 구분 보기
  - XLSX 변환 시 `sheetMap(sheetName/startPage/endPage)` 메타데이터를 생성하도록 보강했습니다.
  - 리본에 `시트 선택` UI를 추가해 시트 선택 시 해당 시트 시작 페이지로 즉시 이동할 수 있습니다.
- 공통 안정화
  - 삭제된 빠른 설정 버튼 참조를 제거해 툴바 상태 갱신 시 오류 가능성을 줄였습니다.
  - 문서 열기 오류 코드 매핑에 `ENGINE_TIMEOUT`/`RENDER_ARTIFACT_DETECTED` 처리를 추가했습니다.
