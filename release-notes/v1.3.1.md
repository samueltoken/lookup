# lookup v1.3.1

## HWP 열기 안정화
- 한글 문서 기본 변환 순서를 `SaveAs PDF` 중심으로 고정했습니다.
- 일반 열기에서는 `인쇄 방식 변환`을 기본으로 쓰지 않도록 분리했습니다.
- 보안 모듈 등록 실패 시에도 앱이 멈추지 않고 호환 보기로 자동 전환되게 했습니다.
- 변환 단계별 점검(열기/저장/검증/종료)을 강화해 한컴 오류창 노출을 줄였습니다.

## HWP 변환 안전성
- 변환된 PDF를 열기 전에 구조 검사를 강화했습니다.
- 손상된 PDF 후보는 캐시에 남기지 않고 다음 변환 방식으로 재시도합니다.
- 사용자에게는 기술 오류 대신 쉬운 안내 문구를 보여주도록 정리했습니다.

## 업데이트 완료 팝업 개선
- 버전(`v1.3.1`)을 변경 항목 목록과 분리해 상단 배지로 표시합니다.
- 섹션 제목과 하위 항목 들여쓰기를 분리해 읽기 쉽게 바꿨습니다.
- HTML/마크다운 기호가 그대로 보이지 않도록 정제 규칙을 강화했습니다.
